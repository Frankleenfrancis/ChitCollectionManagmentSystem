Project Overview
The Chit Collection Management System is a web-based application designed to manage chit fund operations efficiently. The system automates customer management, chit plan creation, monthly collections, payment tracking, and reporting.
It ensures secure authentication, role-based access, and real-time tracking of financial transactions for better transparency and control.

System Architecture
•	The system follows a client-server architecture:
•	Frontend → UI layer (User interaction)
•	Backend API → Business logic layer (Spring Boot)
•	Database → Data storage (MySQL)
•	Security Layer → JWT Authentication & Role-based Access
